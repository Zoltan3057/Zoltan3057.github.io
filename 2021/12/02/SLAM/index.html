5. 激光点云的稳定性检测

在多线激光的选型中，点云的稳定性较为重要。通过将雷达数据打在一面墙上，判断点距离与拟合平面的平均距离，可以判断激光雷达产品点云的稳定性。主要是点云的选取与点云平面的拟合。

In the task of judging the performace of an lidar, the point cloud stability is one curcial criteria. I record a point cloud rosbag by placing the lidar in front of a wall. By draging a small rectange area from the point cloud, the program will detect the plane of this rectange area and select all points in this plane. Then, it will calculate the average distance between points and the plane. By comparing the average distance, we can get a clue about the performance of the test lidar.
